# 服务器部署指南

## 环境要求
- Linux 系统
- g++ 编译器
- PHP 7.0+ 环境
- curl 工具

## 1. C++ 服务器部署

### 编译程序
```bash
g++ time_server.cpp -o server -lcurl
```

### 启动服务器
./server

## 2. PHP 环境配置

### 安装依赖
```bash
sudo apt update
sudo apt install php-curl
```

这里要用ifconfig 查看当前的linux是什么ip
执行ifconfig  把linux当前的ip复制一下 执行这个

启动php服务器 (替换实际IP)
```bash
php -S 192.168.1.175:8080
```
